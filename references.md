# MySQL Employees Sample Database (Reference)

This project references the **MySQL Employees Sample Database** (`test_db`)
for conceptual understanding and example scenarios related to database design
and recovery discussions.

**Source Repository:**  
https://github.com/datacharmer/test_db

**Original Authors:**
- Fusheng Wang
- Carlo Zaniolo
- Giuseppe Maxia
- Patrick Crews

**Description:**
The dataset contains a large, fabricated employee database commonly used
for teaching, testing, and benchmarking database systems.
